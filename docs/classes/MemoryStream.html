    asyncRead

    • asyncRead(buffer: Uint8Array, offset: number, count: number): Promise<any>

      异步读取当前流中的字节。

      参数

      • buffer: Uint8Array

        要读取的字节将存储在此缓冲区中。

      • offset: number

        缓冲区中的从零开始的字节偏移量，从此处开始存储从当前流中读取的数据。

      • count: number

        要从当前流中最多读取的字节数。

      返回 Promise<any>

      返回一个 Promise，该 Promise 在读取完成时解析为读取的字节数，或者在发生错误时解析为错误对象。

      继承自 Stream.asyncRead

      • count: number

      返回 Promise<any>

      返回一个 Promise，该 Promise 在写入完成时解析为写入的字节数，或者在发生错误时解析为错误对象。

    • copyTo(destination: Stream, bufferSize: number): void

      使用指定的缓冲区大小，从当前流中读取字节并将其写入到另一流中。

        当前流的内容将复制到的流。

      • bufferSize: number

        缓冲区的大小。 此值必须大于零。 默认大小为 81920。

    read

    • read(buffer: Uint8Array, offset: number, count: number): number

      从当前流中读取字节块并将数据写入缓冲区。

      参数

      • buffer: Uint8Array

        当此方法返回时，包含指定的字节数组，该数组中从 offset 到 (offset + count -1) 之间的值由从当前流中读取的字符替换。

      • offset: number

        buffer 中的从零开始的字节偏移量，从此处开始存储当前流中的数据。

      • count: number

        最多读取的字节数。

      返回 number

      写入缓冲区中的总字节数。 如果字节数当前不可用，则总字节数可能小于所请求的字节数；如果在读取到任何字节前已到达流结尾，则为零。

      重写了 Stream.read

    readByte

    • readByte(): number

      从当前流中读取一个字节。

      返回 number

      强制转换为 Int32 的字节；或者如果已到达流的末尾，则为 -1。

      重写了 Stream.readByte

    seek

    • seek(offset: number, loc: SeekOrigin): number

      将当前流中的位置设置为指定值。

      参数

        流内的新位置。 它是相对于 loc 参数的位置，而且可正可负。

      • loc: SeekOrigin

        类型 SeekOrigin 的值，它用作查找引用点。

      返回 number

      流内的新位置，通过将初始引用点和偏移量合并计算而得。

      重写了 Stream.seek
