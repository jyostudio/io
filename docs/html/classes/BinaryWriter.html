close

  • close(): void

  • 关闭当前的 BinaryWriter 和基础流。

    返回 void

flush

  • flush(): void

  • 清理当前编写器的所有缓冲区，使所有缓冲数据写入基础设备。

    返回 void

seek

  • seek(offset, origin): Number

  • 设置当前流中的位置。

    参数

    • offset: Number

      相对于 origin 的字节偏移量。

    • origin: SeekOrigin

      SeekOrigin 的一个字段，指示获取新位置所依据的参考点。

    返回 Number

    包含当前流的位置。

    返回 void

  • writeBytes(buffer, index, count): void

  • 将字节数组区域写入当前流。

    参数

    • buffer: Uint8Array

      包含要写入的数据的字节数组。

    • index: Number

      要从 buffer 中读取且要写入流的第一个字节的索引。

    • count: Number

      要从 buffer 中读取且要写入流的字节数。

    返回 void

writeChar

  • writeChar(ch): void

    返回 void

    返回 void

  • writeChars(chars, index, count): void

  • 将字符数组部分写入当前流，并根据所使用的 Encoding（可能还根据向流中写入的特定字符），提升流的当前位置。

    参数

    • chars: List<String>

      包含要写入的数据的字符数组。

    • index: Number

      要从 chars 中读取且要写入流的第一个字符的索引。

    • count: Number

      要从 chars 中读取且要写入流的字符数。

    返回 void
